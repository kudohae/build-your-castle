# T44 — [Phase 4] — Quest Definitions & Check Logic

## Overview
This task implements the complete quest system data layer: it defines all seven quests with their check functions, tracks the singleton `GameState` that accumulates progress events, and implements the `detectArchStructure` algorithm for the complex arch quest that requires spatial reasoning over the block grid.

## Scope
Create `src/systems/quests.ts`. Define `GameState` interface with seven tracked metrics: `totalBlocksCollected`, `maxHeight`, `totalEngravedPixels`, `tidesWeathered`, `collaboratorCount`, `avgArtScore`, `voteCount`. Define `QUESTS` array with all 7 quest objects, each having: `id` (string slug), `title` (Korean), `description` (Korean), `check(state): boolean` function, `reward` (numerical token amount), `promotionCode` (AIT promotion code string). Implement `detectArchStructure(blocks: Block[][]): boolean` which scans the 2D block grid for two vertical columns at least 3 blocks tall, separated by at least 2 empty columns, connected at the top by at least 1 horizontal block with an air gap beneath the connector. Implement `updateGameState(event: GameEvent, data: any): void` which updates the singleton `gameState` based on incoming events from the game loop and block system, then calls `checkAndGrantQuests` from T45.

## Output Files
- `src/systems/quests.ts` — new file; `GameState`, `QUESTS`, `detectArchStructure`, `updateGameState`, singleton `gameState`

### detectArchStructure Algorithm

```typescript
export function detectArchStructure(grid: BlockGrid): boolean {
  if (grid.length === 0 || grid[0].length === 0) return false;

  const rows = grid.length;
  const cols = grid[0].length;

  // For each possible pair of left-column and right-column:
  for (let leftCol = 0; leftCol < cols - 3; leftCol++) {
    for (let rightCol = leftCol + 3; rightCol < cols; rightCol++) {

      // Check gap between columns: at least 2 empty columns
      const gap = rightCol - leftCol - 1;
      if (gap < 2) continue;

      // Check left column: ≥3 vertically contiguous blocks
      const leftHeight = getColumnHeight(grid, rows, leftCol);
      if (leftHeight < 3) continue;

      // Check right column: ≥3 vertically contiguous blocks
      const rightHeight = getColumnHeight(grid, rows, rightCol);
      if (rightHeight < 3) continue;

      // Check horizontal connector at the top of shorter column
      const connectorRow = rows - Math.min(leftHeight, rightHeight) - 1;
      if (connectorRow < 0) continue;

      const hasConnector = hasHorizontalBridge(
        grid, connectorRow, leftCol, rightCol
      );
      if (!hasConnector) continue;

      // Check air gap beneath connector (the row immediately below connector)
      const airRow = connectorRow + 1;
      if (airRow >= rows) continue;

      const hasAirGap = isRowEmptyBetween(grid, airRow, leftCol + 1, rightCol - 1);
      if (hasAirGap) return true;
    }
  }

  return false;
}

function getColumnHeight(grid: BlockGrid, rows: number, col: number): number {
  let height = 0;
  // Count from bottom up
  for (let row = rows - 1; row >= 0; row--) {
    if (grid[row][col] !== null) height++;
    else break; // must be contiguous from ground
  }
  return height;
}

function hasHorizontalBridge(
  grid: BlockGrid,
  row: number,
  leftCol: number,
  rightCol: number
): boolean {
  // At least one block must exist at row level between the two columns
  for (let col = leftCol; col <= rightCol; col++) {
    if (grid[row][col] !== null) return true;
  }
  return false;
}

function isRowEmptyBetween(
  grid: BlockGrid,
  row: number,
  fromCol: number,
  toCol: number
): boolean {
  for (let col = fromCol; col <= toCol; col++) {
    if (grid[row][col] !== null) return false;
  }
  return true;
}
```

The algorithm is O(cols² × rows) — for typical castle grids (100×60 blocks) this is ~600,000 operations worst case, acceptable for an event-driven check (not per-frame).

### GameState Singleton & updateGameState

```typescript
export const gameState: GameState = {
  totalBlocksCollected: 0,
  maxHeight: 0,
  totalEngravedPixels: 0,
  tidesWeathered: 0,
  collaboratorCount: 0,
  avgArtScore: 0,
  voteCount: 0,
};

/** Called by game systems whenever a relevant action occurs. */
export function updateGameState(
  event: GameEvent,
  currentBlocks?: BlockGrid
): void {
  switch (event.type) {
    case 'block_collected':
      gameState.totalBlocksCollected += (event.data.count as number) ?? 1;
      break;

    case 'block_placed':
      // Recalculate max tower height after placement
      if (currentBlocks) {
        gameState.maxHeight = calculateMaxHeight(currentBlocks);
      }
      break;

    case 'block_carved':
      gameState.totalEngravedPixels += (event.data.pixels as number) ?? 0;
      break;

    case 'tide_survived':
      gameState.tidesWeathered += 1;
      break;

    case 'collaborator_added':
      // Use a Set externally to avoid counting the same collaborator twice
      gameState.collaboratorCount = (event.data.totalCount as number) ?? gameState.collaboratorCount;
      break;

    case 'score_updated':
      gameState.avgArtScore = (event.data.avgScore as number) ?? gameState.avgArtScore;
      gameState.voteCount   = (event.data.voteCount as number) ?? gameState.voteCount;
      break;
  }

  // T45 will call checkAndGrantQuests here — imported lazily to avoid circular deps
  import('../ait/promotion').then(({ checkAndGrantQuests }) => {
    checkAndGrantQuests(gameState, currentBlocks);
  });
}

function calculateMaxHeight(grid: BlockGrid): number {
  const cols = grid[0]?.length ?? 0;
  let max = 0;
  for (let col = 0; col < cols; col++) {
    max = Math.max(max, getColumnHeight(grid, grid.length, col));
  }
  return max;
}
```

### Hydrating GameState from DB on App Start

```typescript
export async function hydrateGameState(plotId: string): Promise<void> {
  const { supabase } = await import('../supabase/client');

  const { data } = await supabase
    .from('plot_game_state')
    .select('*')
    .eq('plot_id', plotId)
    .maybeSingle();

  if (data) {
    gameState.totalBlocksCollected = data.blocks_collected ?? 0;
    gameState.maxHeight            = data.max_height ?? 0;
    gameState.totalEngravedPixels  = data.engraved_pixels ?? 0;
    gameState.tidesWeathered       = data.tides_weathered ?? 0;
    gameState.collaboratorCount    = data.collaborator_count ?? 0;
  }
}
```

The `plot_game_state` table is a companion table to `plots` updated periodically by the game loop (not every event to avoid excessive writes).

## Key Constraints
- `detectArchStructure` checks for contiguous columns from ground up — floating pillars do not count.
- The arch gap must be at least 2 clear block columns, meaning the arch must span at least a 4-block-wide opening (left col + 2 gap + right col).
- `updateGameState` uses a dynamic `import()` for `checkAndGrantQuests` to prevent a circular dependency (quests.ts → promotion.ts → quests.ts).
- `gameState` is a singleton; it is NOT reset between sessions — it represents the cumulative lifetime progress of the player's plot.
- The `tower` quest requires height ≥ 20 blocks (20 × 32px = 640px) measured as contiguous vertical blocks in a single column, not total blocks placed.
- `collaboratorCount` tracks the total distinct users ever invited, not the current active count — a player who removes all collaborators still counts toward their own `collab` quest progress if they previously had 3.
- The spatial check for arch is triggered on `block_placed` events only (not every frame) to avoid performance impact.

## Acceptance Criteria
- [ ] `QUESTS` array contains exactly 7 entries with non-empty `id`, `title`, `description`, `promotionCode`, and `reward > 0`.
- [ ] `gameState.totalBlocksCollected` increments by 1 for each `block_collected` event with default count.
- [ ] `gameState.maxHeight` is recalculated correctly after block placement and matches the tallest contiguous column.
- [ ] `detectArchStructure` returns `true` for a valid arch (2 columns ≥3 tall, ≥2 gap, top connector, air beneath).
- [ ] `detectArchStructure` returns `false` if the gap between columns is only 1 block wide.
- [ ] `detectArchStructure` returns `false` if there is no air gap beneath the connector block.
- [ ] The `high_art` quest `check` returns `false` when `voteCount < 3`, even if `avgArtScore >= 4.5`.
- [ ] `hydrateGameState` correctly populates all `gameState` fields from the `plot_game_state` DB row on app start.
