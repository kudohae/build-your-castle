# T36 — [Phase 3] — Guestbook UI

## Overview
This task builds the interactive guestbook panel that slides up from the bottom of the screen when a player visits another player's castle plot. It renders the message list, handles input with a live character counter, and gives the plot owner per-message delete controls with optimistic UI updates.

## Scope
Create `src/ui/guestbook.ts` as a self-contained Canvas 2D overlay panel. The panel is hidden by default and becomes visible when the local player enters a foreign plot zone. It implements: a slide-up animation (translateY from +300px to 0 over 250ms easing `ease-out`); a text input field rendered via a DOM `<textarea>` element overlaid on the canvas; a real-time character counter ("N / 50") that turns red when ≥ 45 characters; a "등록" submit button; a scrollable message list drawn on canvas showing nickname, message text, and relative timestamp (e.g. "3분 전"); and per-row 🗑️ delete buttons visible only to the plot owner. The panel communicates with T35 functions for all DB operations. On submit success, the new entry is prepended to the local list and the panel scrolls to top. On delete, the row is removed from the local array immediately (optimistic update) and the DB call fires asynchronously — on DB failure the row is re-inserted and an error toast is shown.

## Output Files
- `src/ui/guestbook.ts` — new file; complete guestbook panel implementation

## Depends On
- T35 — provides `addMessage`, `getMessages`, `deleteMessage`, and `GuestbookEntry` type

```typescript
// src/ui/guestbook.ts

import { addMessage, deleteMessage, getMessages, GuestbookEntry, GuestbookError } from '../supabase/guestbook';

interface GuestbookPanelState {
  isOpen: boolean;
  plotId: string | null;
  plotOwnerId: string | null;
  currentUserId: string | null;
  isOwner: boolean;
  entries: GuestbookEntry[];
  scrollOffset: number;       // pixels scrolled from top
  slideY: number;             // current slide animation Y (0 = fully open)
  isAnimating: boolean;
  inputText: string;
  isSubmitting: boolean;
}

const PANEL_HEIGHT = 320;
const PANEL_PADDING = 16;
const ROW_HEIGHT = 60;
const INPUT_HEIGHT = 44;

export class GuestbookPanel {
  private state: GuestbookPanelState = {
    isOpen: false,
    plotId: null,
    plotOwnerId: null,
    currentUserId: null,
    isOwner: false,
    entries: [],
    scrollOffset: 0,
    slideY: PANEL_HEIGHT,
    isAnimating: false,
    inputText: '',
    isSubmitting: false,
  };

  private textarea: HTMLTextAreaElement | null = null;
  private canvas: HTMLCanvasElement;
  private ctx: CanvasRenderingContext2D;

  constructor(canvas: HTMLCanvasElement, ctx: CanvasRenderingContext2D) {
    this.canvas = canvas;
    this.ctx = ctx;
  }
```

### Open & Close with Slide Animation

```typescript
  async open(plotId: string, plotOwnerId: string, currentUserId: string): Promise<void> {
    this.state.plotId = plotId;
    this.state.plotOwnerId = plotOwnerId;
    this.state.currentUserId = currentUserId;
    this.state.isOwner = plotOwnerId === currentUserId;
    this.state.isOpen = true;
    this.state.scrollOffset = 0;

    // Load messages before animating in
    try {
      this.state.entries = await getMessages(plotId);
    } catch {
      this.state.entries = [];
    }

    this.createTextareaOverlay();
    this.animateSlide('open');
  }

  close(): void {
    this.animateSlide('close', () => {
      this.state.isOpen = false;
      this.state.plotId = null;
      this.destroyTextareaOverlay();
    });
  }

  private animateSlide(direction: 'open' | 'close', onComplete?: () => void): void {
    const target = direction === 'open' ? 0 : PANEL_HEIGHT;
    const start = this.state.slideY;
    const duration = 250;
    const startTime = performance.now();

    const step = (now: number) => {
      const t = Math.min((now - startTime) / duration, 1);
      const eased = 1 - Math.pow(1 - t, 3); // ease-out cubic
      this.state.slideY = start + (target - start) * eased;
      if (t < 1) {
        requestAnimationFrame(step);
      } else {
        this.state.slideY = target;
        onComplete?.();
      }
    };
    requestAnimationFrame(step);
  }
```

### DOM Textarea Overlay for Input

```typescript
  private createTextareaOverlay(): void {
    const ta = document.createElement('textarea');
    ta.maxLength = 50;
    ta.placeholder = '방명록을 남겨보세요...';
    ta.rows = 1;
    ta.style.cssText = `
      position: absolute;
      bottom: ${PANEL_HEIGHT - this.state.slideY + INPUT_HEIGHT + PANEL_PADDING * 2}px;
      left: ${PANEL_PADDING}px;
      width: calc(100% - ${PANEL_PADDING * 2 + 72}px);
      height: ${INPUT_HEIGHT}px;
      border-radius: 8px;
      border: 1px solid #ccc;
      padding: 8px;
      font-size: 14px;
      resize: none;
      z-index: 100;
    `;
    ta.addEventListener('input', () => {
      this.state.inputText = ta.value;
    });
    document.body.appendChild(ta);
    this.textarea = ta;
  }

  private destroyTextareaOverlay(): void {
    this.textarea?.remove();
    this.textarea = null;
  }
```

### Draw Method — Panel Background, List, Controls

```typescript
  draw(): void {
    if (!this.state.isOpen && this.state.slideY >= PANEL_HEIGHT) return;

    const { ctx, canvas } = this;
    const panelTop = canvas.height - PANEL_HEIGHT + this.state.slideY;
    const panelWidth = canvas.width;

    // Panel background
    ctx.fillStyle = 'rgba(255, 250, 235, 0.97)';
    ctx.beginPath();
    ctx.roundRect(0, panelTop, panelWidth, PANEL_HEIGHT, [16, 16, 0, 0]);
    ctx.fill();

    // Header
    ctx.fillStyle = '#5c3d1e';
    ctx.font = 'bold 16px Pretendard, sans-serif';
    ctx.fillText('📖 방명록', PANEL_PADDING, panelTop + 28);

    // Close button
    ctx.fillStyle = '#999';
    ctx.font = '20px sans-serif';
    ctx.fillText('✕', panelWidth - 36, panelTop + 28);

    // Character counter
    const count = this.state.inputText.length;
    ctx.font = `12px Pretendard, sans-serif`;
    ctx.fillStyle = count >= 45 ? '#e53e3e' : '#999';
    ctx.fillText(`${count} / 50`, panelWidth - PANEL_PADDING - 80, panelTop + 60);

    // Submit button
    ctx.fillStyle = this.state.isSubmitting ? '#ccc' : '#f5a623';
    ctx.beginPath();
    ctx.roundRect(panelWidth - PANEL_PADDING - 60, panelTop + 44, 60, INPUT_HEIGHT, 8);
    ctx.fill();
    ctx.fillStyle = '#fff';
    ctx.font = 'bold 14px Pretendard, sans-serif';
    ctx.fillText('등록', panelWidth - PANEL_PADDING - 40, panelTop + 70);

    // Message list (clipped)
    const listTop = panelTop + 100;
    const listHeight = PANEL_HEIGHT - 110;
    ctx.save();
    ctx.beginPath();
    ctx.rect(0, listTop, panelWidth, listHeight);
    ctx.clip();

    this.state.entries.forEach((entry, i) => {
      const rowY = listTop + i * ROW_HEIGHT - this.state.scrollOffset;
      if (rowY + ROW_HEIGHT < listTop || rowY > listTop + listHeight) return;

      ctx.fillStyle = i % 2 === 0 ? 'rgba(0,0,0,0.02)' : 'transparent';
      ctx.fillRect(0, rowY, panelWidth, ROW_HEIGHT);

      ctx.fillStyle = '#5c3d1e';
      ctx.font = 'bold 13px Pretendard, sans-serif';
      ctx.fillText(entry.authorNickname, PANEL_PADDING, rowY + 20);

      ctx.fillStyle = '#333';
      ctx.font = '13px Pretendard, sans-serif';
      ctx.fillText(entry.message, PANEL_PADDING, rowY + 40);

      ctx.fillStyle = '#aaa';
      ctx.font = '11px Pretendard, sans-serif';
      ctx.fillText(this.formatRelativeTime(entry.createdAt), panelWidth - 100, rowY + 20);

      if (this.state.isOwner) {
        ctx.font = '16px sans-serif';
        ctx.fillText('🗑️', panelWidth - PANEL_PADDING - 20, rowY + 32);
      }
    });

    ctx.restore();
  }
```

### Submit & Delete Handlers

```typescript
  async handleSubmit(): Promise<void> {
    if (this.state.isSubmitting || !this.state.plotId || !this.state.currentUserId) return;
    const message = this.state.inputText.trim();
    if (!message) return;

    this.state.isSubmitting = true;
    try {
      const entry = await addMessage(this.state.plotId, this.state.currentUserId, message);
      this.state.entries.unshift(entry);        // prepend new entry
      if (this.state.entries.length > 20) this.state.entries.pop();
      this.state.scrollOffset = 0;              // scroll to top
      this.state.inputText = '';
      if (this.textarea) this.textarea.value = '';
    } catch (err) {
      const msg = err instanceof GuestbookError ? err.message : '등록에 실패했습니다.';
      this.showToast(msg, 'error');
    } finally {
      this.state.isSubmitting = false;
    }
  }

  async handleDeleteEntry(entryId: string): Promise<void> {
    if (!this.state.currentUserId || !this.state.isOwner) return;
    const idx = this.state.entries.findIndex(e => e.id === entryId);
    if (idx === -1) return;

    // Optimistic removal
    const [removed] = this.state.entries.splice(idx, 1);

    try {
      await deleteMessage(entryId, this.state.currentUserId);
    } catch {
      // Rollback on failure
      this.state.entries.splice(idx, 0, removed);
      this.showToast('삭제에 실패했습니다.', 'error');
    }
  }
```

### Touch/Click Hit-Testing

```typescript
  handlePointerDown(x: number, y: number): boolean {
    if (!this.state.isOpen) return false;
    const panelTop = this.canvas.height - PANEL_HEIGHT + this.state.slideY;

    // Close button
    if (x > this.canvas.width - 46 && y > panelTop + 10 && y < panelTop + 40) {
      this.close();
      return true;
    }

    // Submit button
    const btnX = this.canvas.width - PANEL_PADDING - 60;
    const btnY = this.canvas.height - PANEL_HEIGHT + this.state.slideY + 44;
    if (x > btnX && y > btnY && y < btnY + INPUT_HEIGHT) {
      this.handleSubmit();
      return true;
    }

    // Delete buttons per entry row
    if (this.state.isOwner) {
      const listTop = panelTop + 100;
      this.state.entries.forEach((entry, i) => {
        const rowY = listTop + i * ROW_HEIGHT - this.state.scrollOffset;
        const delX = this.canvas.width - PANEL_PADDING - 30;
        if (x > delX && y > rowY && y < rowY + ROW_HEIGHT) {
          this.handleDeleteEntry(entry.id);
        }
      });
    }

    return y > panelTop; // consume event if within panel
  }

  handleScroll(deltaY: number): void {
    if (!this.state.isOpen) return;
    const maxScroll = Math.max(0, this.state.entries.length * ROW_HEIGHT - 160);
    this.state.scrollOffset = Math.max(0, Math.min(this.state.scrollOffset + deltaY, maxScroll));
  }

  private formatRelativeTime(iso: string): string {
    const diff = Date.now() - new Date(iso).getTime();
    const minutes = Math.floor(diff / 60000);
    if (minutes < 1) return '방금';
    if (minutes < 60) return `${minutes}분 전`;
    const hours = Math.floor(minutes / 60);
    if (hours < 24) return `${hours}시간 전`;
    return `${Math.floor(hours / 24)}일 전`;
  }

  private showToast(msg: string, type: 'success' | 'error'): void {
    // Delegates to global toast system (implemented separately)
    (window as any).__sandCastleToast?.(msg, type);
  }
}
```

## Key Constraints
- The panel must NOT render when the player is on their own plot — `isOwner` still shows the delete buttons but the "입장" trigger for the panel only fires for foreign plots.
- The DOM `<textarea>` must be destroyed when the panel closes to avoid ghost input elements accumulating.
- Optimistic delete must fully rollback (both array re-insertion and index preservation) on network failure.
- Scroll state resets to 0 whenever the panel is opened for a new plot.
- The character counter color change threshold is 45/50, not 50/50, giving the user a warning before hitting the limit.
- The panel draw call must be invoked from the main render loop after all world elements are drawn but before HUD elements.
- Touch events must return `true` (consumed) whenever the pointer lands inside the panel boundary to prevent world-click handlers from firing.

## Acceptance Criteria
- [ ] The panel slides up from the bottom with a visible ease-out animation (250ms) when `open()` is called.
- [ ] The character counter displays "N / 50" and turns red at 45 characters.
- [ ] Submitting a valid message prepends it to the top of the list and clears the input without a page reload.
- [ ] Submitting a message longer than 50 characters (bypassing `maxLength`) shows an error toast from `GuestbookError`.
- [ ] The 🗑️ delete button is visible on each row only when `isOwner === true`.
- [ ] Clicking 🗑️ removes the row instantly from the UI; if the DB call fails the row reappears and an error toast fires.
- [ ] Scrolling within the message list scrolls only the list content, not the page.
- [ ] Clicking ✕ or pressing outside the panel closes it with the reverse slide animation.
