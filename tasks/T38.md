# T38 — [Phase 3] — Castle Tour Mode UI Overlay

## Overview
This task extends the `tourMode.ts` file created in T37 by adding the Canvas 2D UI overlay drawn on top of the game world during an active tour. It renders contextual information about the current plot, navigation controls, and integrates the star rating widget from T32 so users can rate a plot without leaving tour mode.

## Scope
Modify `src/ui/tourMode.ts` to add a `TourOverlay` class (or extend `TourController`) with a `draw(ctx, canvas)` method called every frame when `isActive === true`. The overlay renders: a "🏰 성 투어 모드" banner centered at the top with a semi-transparent dark background pill; an "N / 10" progress indicator directly below the banner; a bottom card showing the current plot owner's nickname and `avgScore` as filled stars (e.g. ★★★★☆); a "⭐ 별점 주기" button that opens the `StarRatingWidget` from T32 for the current plot; a "✕ 투어 종료" button in the top-right corner. Canvas taps outside all interactive elements call `tourController.stopTour()`. Between plot transitions, a black fade (opacity 0→0.6→0 over 400ms) masks the camera tween. All text uses Korean labels and the Pretendard font stack.

## Output Files
- `src/ui/tourMode.ts (modify)` — adds `TourOverlay` class and fade-transition logic to the existing file from T37

## Depends On
- T37 — provides `TourController`, `TourState`, `TourPlot`, and `tourController` singleton
- T32 — provides `StarRatingWidget` for in-tour rating (opens as a modal overlay above the tour UI)

### Fade Transition State

```typescript
// Append to src/ui/tourMode.ts (after T37 code)

import { StarRatingWidget } from './starRating'; // T32

interface FadeState {
  opacity: number;        // 0.0–1.0
  direction: 'in' | 'out' | 'none';
  startTime: number;
}

const FADE_DURATION_MS = 200; // half-cycle (fade-out then fade-in)
```

### TourOverlay Class

```typescript
export class TourOverlay {
  private fade: FadeState = { opacity: 0, direction: 'none', startTime: 0 };
  private ratingWidget: StarRatingWidget | null = null;

  constructor(private controller: TourController) {}

  /** Called by the main render loop every frame. */
  draw(ctx: CanvasRenderingContext2D, canvas: HTMLCanvasElement): void {
    const state = this.controller.getState();
    if (!state.isActive) return;

    this.updateFade();

    const plot = state.topPlots[state.currentIndex];
    if (!plot) return;

    const W = canvas.width;
    const H = canvas.height;

    // --- Banner (top center) ---
    this.drawBanner(ctx, W, state);

    // --- Bottom info card ---
    this.drawInfoCard(ctx, W, H, plot);

    // --- Fade overlay (drawn on top of everything during transitions) ---
    if (this.fade.opacity > 0) {
      ctx.fillStyle = `rgba(0, 0, 0, ${this.fade.opacity})`;
      ctx.fillRect(0, 0, W, H);
    }

    // --- Rating widget (drawn last so it is on top of fade) ---
    this.ratingWidget?.draw(ctx, canvas);
  }

  private drawBanner(ctx: CanvasRenderingContext2D, W: number, state: TourState): void {
    const bannerText = '🏰 성 투어 모드';
    const progressText = `${state.currentIndex + 1} / ${state.topPlots.length}`;

    // Pill background
    ctx.fillStyle = 'rgba(0, 0, 0, 0.60)';
    ctx.beginPath();
    ctx.roundRect(W / 2 - 120, 12, 240, 36, 18);
    ctx.fill();

    // Banner text
    ctx.fillStyle = '#fff';
    ctx.font = 'bold 15px Pretendard, sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText(bannerText, W / 2, 36);

    // Progress indicator below banner
    ctx.fillStyle = 'rgba(0, 0, 0, 0.45)';
    ctx.beginPath();
    ctx.roundRect(W / 2 - 40, 54, 80, 24, 12);
    ctx.fill();

    ctx.fillStyle = '#ffd700';
    ctx.font = '13px Pretendard, sans-serif';
    ctx.fillText(progressText, W / 2, 70);

    // Close button (top-right)
    ctx.fillStyle = 'rgba(0,0,0,0.55)';
    ctx.beginPath();
    ctx.roundRect(W - 52, 12, 40, 36, 8);
    ctx.fill();
    ctx.fillStyle = '#fff';
    ctx.font = '18px sans-serif';
    ctx.fillText('✕', W - 32, 36);

    ctx.textAlign = 'left'; // reset
  }

  private drawInfoCard(
    ctx: CanvasRenderingContext2D,
    W: number,
    H: number,
    plot: TourPlot
  ): void {
    const cardH = 96;
    const cardY = H - cardH - 16;
    const cardX = 16;
    const cardW = W - 32;

    // Card background
    ctx.fillStyle = 'rgba(255, 250, 235, 0.92)';
    ctx.beginPath();
    ctx.roundRect(cardX, cardY, cardW, cardH, 12);
    ctx.fill();

    // Owner nickname
    ctx.fillStyle = '#5c3d1e';
    ctx.font = 'bold 16px Pretendard, sans-serif';
    ctx.fillText(`${plot.ownerNickname}의 성`, cardX + 16, cardY + 28);

    // Star rating display
    const stars = this.renderStars(plot.avgScore);
    ctx.font = '20px sans-serif';
    ctx.fillText(stars, cardX + 16, cardY + 58);

    ctx.fillStyle = '#888';
    ctx.font = '12px Pretendard, sans-serif';
    ctx.fillText(
      `${plot.avgScore.toFixed(1)}점 (${plot.voteCount}명 투표)`,
      cardX + 16,
      cardY + 78
    );

    // "별점 주기" button
    const btnW = 110;
    const btnX = cardX + cardW - btnW - 16;
    ctx.fillStyle = '#f5a623';
    ctx.beginPath();
    ctx.roundRect(btnX, cardY + 28, btnW, 36, 8);
    ctx.fill();
    ctx.fillStyle = '#fff';
    ctx.font = 'bold 13px Pretendard, sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText('⭐ 별점 주기', btnX + btnW / 2, cardY + 51);
    ctx.textAlign = 'left';
  }

  private renderStars(score: number): string {
    const full  = Math.floor(score);
    const half  = score - full >= 0.5 ? 1 : 0;
    const empty = 5 - full - half;
    return '★'.repeat(full) + (half ? '½' : '') + '☆'.repeat(empty);
  }
```

### Transition Fade Logic

```typescript
  /** Call this before each cycle advance to trigger the fade. */
  triggerTransitionFade(onMidpoint: () => void): void {
    this.fade = { opacity: 0, direction: 'in', startTime: performance.now() };
    // At full opacity (midpoint), execute the camera tween start
    setTimeout(onMidpoint, FADE_DURATION_MS);
  }

  private updateFade(): void {
    if (this.fade.direction === 'none') return;

    const elapsed = performance.now() - this.fade.startTime;
    const t = Math.min(elapsed / FADE_DURATION_MS, 1);

    if (this.fade.direction === 'in') {
      this.fade.opacity = t * 0.6;
      if (t >= 1) {
        this.fade.direction = 'out';
        this.fade.startTime = performance.now();
      }
    } else if (this.fade.direction === 'out') {
      this.fade.opacity = 0.6 * (1 - t);
      if (t >= 1) {
        this.fade.opacity = 0;
        this.fade.direction = 'none';
      }
    }
  }
```

### Pointer Hit-Testing — stopTour & Rating Widget

```typescript
  handlePointerDown(
    x: number,
    y: number,
    canvas: HTMLCanvasElement
  ): boolean {
    const state = this.controller.getState();
    if (!state.isActive) return false;

    // Delegate to rating widget first (it is on top)
    if (this.ratingWidget?.handlePointerDown(x, y)) return true;

    const W = canvas.width;
    const H = canvas.height;

    // Close button hit area
    if (x > W - 52 && x < W - 12 && y > 12 && y < 48) {
      this.controller.stopTour();
      return true;
    }

    // "별점 주기" button hit area
    const plot = state.topPlots[state.currentIndex];
    if (plot) {
      const cardX = 16;
      const cardY = H - 96 - 16;
      const cardW = W - 32;
      const btnW = 110;
      const btnX = cardX + cardW - btnW - 16;
      if (x > btnX && x < btnX + btnW && y > cardY + 28 && y < cardY + 64) {
        this.openRatingWidget(plot, canvas);
        return true;
      }
    }

    // Tap outside all UI elements → stop tour
    this.controller.stopTour();
    return true;
  }

  private openRatingWidget(plot: TourPlot, canvas: HTMLCanvasElement): void {
    this.ratingWidget = new StarRatingWidget({
      plotId: plot.plotId,
      canvas,
      onClose: () => { this.ratingWidget = null; },
    });
  }
}

// Export singleton overlay bound to the shared controller
export const tourOverlay = new TourOverlay(tourController);
```

### Integration with Main Render Loop

The caller (main game loop) must call `tourOverlay.draw(ctx, canvas)` at the end of the render pass, after world and HUD elements:

```typescript
// In main render loop (src/canvas/renderer.ts or similar):
function renderFrame() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  drawWorld(ctx);
  drawHUD(ctx);
  tourOverlay.draw(ctx, canvas);   // always last
}
```

And route pointer events:

```typescript
canvas.addEventListener('pointerdown', (e) => {
  const consumed = tourOverlay.handlePointerDown(e.clientX, e.clientY, canvas);
  if (!consumed) handleWorldPointerDown(e);
});
```

## Key Constraints
- The overlay draw call must be placed after HUD rendering so it occludes the HUD during tour mode.
- The fade effect uses two sequential `FADE_DURATION_MS` windows (fade-to-black, then fade-from-black), not a simultaneous blend.
- Any canvas tap that is not on a recognized interactive element must stop the tour — this prevents users from accidentally building while in tour mode.
- The `StarRatingWidget` from T32 is opened as a subordinate overlay, not a new screen; it must render above the tour overlay itself.
- The "✕ 투어 종료" button and "⭐ 별점 주기" hit targets must have at least 44×44px touch area for accessibility.
- Star display uses ★/½/☆ characters; do not use emoji stars here as rendering varies across OS.
- `tourOverlay.draw()` is a pure draw call; it must not trigger async operations directly — only event handlers may trigger `stopTour()` or open rating widgets.
