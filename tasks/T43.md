# T43 — [Phase 4] — Store UI Screen

## Overview
This task builds the full-screen in-game store overlay, providing a polished shopping interface for all 11 purchasable products organized into five category tabs. It integrates with the payment system from T42 and shows real-time owned-state badges for non-consumable upgrades.

## Scope
Create `src/ui/store.ts` as a full-screen Canvas 2D overlay triggered by a "상점" button in the HUD. The screen has five category tabs rendered as a horizontal tab bar: 도구 업그레이드 (shovel levels, bucket), 구역 (plot expand), 블럭 (wood_plank_pack, stone_pack), 인벤토리 (inventory_expand), 시즌 (season-specific blocks from T50). Each tab renders a grid of product cards (2 columns on mobile, 3 on desktop). Each card shows: an item icon (emoji or colored 32×32 block preview), a product name, a two-line Korean description, and a price label (Korean won format e.g. "₩1,200"). The buy button calls `purchase(productId)` and shows an inline loading spinner while the SDK call is in progress, then shows a "✓ 구매 완료" success state or a red error toast. Non-consumable products already owned show a gray "보유 중" badge instead of the buy button. A "닫기" button in the top right dismisses the overlay. The season tab is pre-selected when opened from the seasonal menu shortcut.

## Output Files
- `src/ui/store.ts` — new file; complete store overlay implementation

## Depends On
- T42 — provides `applyPurchase`, `purchase`, `PRODUCTS`, `PurchaseError`, and the inventory module with owned-state queries

### Store State & Class

```typescript
interface CardState {
  purchasing: boolean;
  owned: boolean;
  justPurchased: boolean;
}

export class StoreScreen {
  private isOpen = false;
  private activeTab: TabId = '도구 업그레이드';
  private cardStates = new Map<ProductId, CardState>();
  private scrollOffset = 0;
  private openedFromSeasonalMenu = false;

  constructor(
    private canvas: HTMLCanvasElement,
    private ctx: CanvasRenderingContext2D
  ) {
    for (const card of PRODUCT_CARDS) {
      this.cardStates.set(card.productId, {
        purchasing: false,
        owned: false,
        justPurchased: false,
      });
    }
  }

  async open(initialTab?: TabId): Promise<void> {
    this.isOpen = true;
    this.activeTab = initialTab ?? '도구 업그레이드';
    this.scrollOffset = 0;
    await this.refreshOwnedStates();
  }

  close(): void {
    this.isOpen = false;
  }

  private async refreshOwnedStates(): Promise<void> {
    // Non-consumable "owned" check — shovel level, bucket slots, plot width
    const shovelLevel = inventory.getShovelLevel();
    this.setOwned(PRODUCTS.shovel_lv2, shovelLevel >= 2);
    this.setOwned(PRODUCTS.shovel_lv3, shovelLevel >= 3);
    this.setOwned(PRODUCTS.shovel_lv5, shovelLevel >= 5);
    this.setOwned(PRODUCTS.bucket_extra, inventory.getBucketSlots() >= 2);
    // plot_expand and inventory_expand are always re-purchasable in this design
  }

  private setOwned(productId: ProductId, owned: boolean): void {
    const state = this.cardStates.get(productId);
    if (state) state.owned = owned;
  }
```

```typescript
  draw(): void {
    if (!this.isOpen) return;
    const { ctx, canvas } = this;
    const W = canvas.width;
    const H = canvas.height;

    // Background overlay
    ctx.fillStyle = 'rgba(30, 20, 10, 0.92)';
    ctx.fillRect(0, 0, W, H);

    // Header
    ctx.fillStyle = '#f5a623';
    ctx.font = 'bold 22px Pretendard, sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText('🏪 상점', W / 2, 44);
    ctx.textAlign = 'left';

    // Close button
    ctx.fillStyle = 'rgba(255,255,255,0.15)';
    ctx.beginPath();
    ctx.roundRect(W - 52, 14, 40, 36, 8);
    ctx.fill();
    ctx.fillStyle = '#fff';
    ctx.font = '20px sans-serif';
    ctx.fillText('✕', W - 38, 38);

    // Tab bar
    this.drawTabs();

    // Product grid
    this.drawProductGrid();
  }

  private drawTabs(): void {
    const tabs: TabId[] = ['도구 업그레이드', '구역', '블럭', '인벤토리', '시즌'];
    const { ctx, canvas } = this;
    const tabW = canvas.width / tabs.length;
    const tabY = 60;

    tabs.forEach((tab, i) => {
      const x = i * tabW;
      const isActive = tab === this.activeTab;

      ctx.fillStyle = isActive ? '#f5a623' : 'rgba(255,255,255,0.08)';
      ctx.fillRect(x, tabY, tabW, 40);

      ctx.fillStyle = isActive ? '#1a0f00' : '#ccc';
      ctx.font = `${isActive ? 'bold ' : ''}12px Pretendard, sans-serif`;
      ctx.textAlign = 'center';
      ctx.fillText(tab, x + tabW / 2, tabY + 26);
    });
    ctx.textAlign = 'left';
  }

  private drawProductGrid(): void {
    const { ctx, canvas } = this;
    const cards = PRODUCT_CARDS.filter(c => c.tab === this.activeTab);
    const cols = canvas.width >= 600 ? 3 : 2;
    const cardW = (canvas.width - 32 - (cols - 1) * 12) / cols;
    const cardH = 140;
    const startY = 116;

    ctx.save();
    ctx.beginPath();
    ctx.rect(0, startY, canvas.width, canvas.height - startY);
    ctx.clip();

    cards.forEach((card, i) => {
      const col = i % cols;
      const row = Math.floor(i / cols);
      const x = 16 + col * (cardW + 12);
      const y = startY + row * (cardH + 12) - this.scrollOffset;

      this.drawCard(card, x, y, cardW, cardH);
    });

    ctx.restore();
  }

  private drawCard(
    card: ProductCard,
    x: number,
    y: number,
    w: number,
    h: number
  ): void {
    const { ctx } = this;
    const state = this.cardStates.get(card.productId)!;

    // Card background
    ctx.fillStyle = state.justPurchased
      ? 'rgba(100, 200, 100, 0.2)'
      : 'rgba(255, 250, 235, 0.10)';
    ctx.beginPath();
    ctx.roundRect(x, y, w, h, 10);
    ctx.fill();

    // Icon
    ctx.font = '28px sans-serif';
    ctx.fillText(card.icon, x + 10, y + 36);

    // Name
    ctx.fillStyle = '#fff';
    ctx.font = 'bold 13px Pretendard, sans-serif';
    ctx.fillText(card.name, x + 10, y + 56);

    // Description (two lines)
    ctx.fillStyle = '#bbb';
    ctx.font = '11px Pretendard, sans-serif';
    const descLines = card.description.split('\n');
    descLines.forEach((line, li) => {
      ctx.fillText(line, x + 10, y + 72 + li * 15);
    });

    // Price / owned / buy button
    const btnY = y + h - 36;
    const btnW = w - 20;

    if (!card.isConsumable && state.owned) {
      // "보유 중" gray badge
      ctx.fillStyle = 'rgba(150,150,150,0.4)';
      ctx.beginPath();
      ctx.roundRect(x + 10, btnY, btnW, 28, 6);
      ctx.fill();
      ctx.fillStyle = '#aaa';
      ctx.font = '12px Pretendard, sans-serif';
      ctx.textAlign = 'center';
      ctx.fillText('보유 중', x + 10 + btnW / 2, btnY + 18);
    } else if (state.purchasing) {
      // Loading state
      ctx.fillStyle = 'rgba(245,166,35,0.5)';
      ctx.beginPath();
      ctx.roundRect(x + 10, btnY, btnW, 28, 6);
      ctx.fill();
      ctx.fillStyle = '#fff';
      ctx.font = '12px Pretendard, sans-serif';
      ctx.textAlign = 'center';
      ctx.fillText('결제 중...', x + 10 + btnW / 2, btnY + 18);

      ctx.textAlign = 'center';

    ctx.textAlign = 'left';
```

### Purchase Handler

```typescript
  async handleBuyCard(productId: ProductId): Promise<void> {
    const state = this.cardStates.get(productId);
    if (!state || state.purchasing || (!PRODUCT_CARDS.find(c => c.productId === productId)?.isConsumable && state.owned)) return;

    state.purchasing = true;
    try {
      await purchase(productId);
      state.justPurchased = true;
      if (!PRODUCT_CARDS.find(c => c.productId === productId)?.isConsumable) {
        state.owned = true;
      }
      // Flash success state for 2 seconds
      setTimeout(() => { state.justPurchased = false; }, 2000);
      this.showToast('구매가 완료되었습니다! 🎉', 'success');
    } catch (err) {
      const msg = err instanceof PurchaseError && err.code === 'USER_CANCELLED'
        ? '결제가 취소되었습니다.'
        : err instanceof PurchaseError
          ? err.message
          : '구매에 실패했습니다.';
      this.showToast(msg, 'error');
    } finally {
      state.purchasing = false;
    }
  }

  handlePointerDown(px: number, py: number): boolean {
    if (!this.isOpen) return false;

    // Close button
    if (px > this.canvas.width - 52 && py > 14 && py < 50) {
      this.close();
      return true;
    }

    // Tab clicks
    const tabs: TabId[] = ['도구 업그레이드', '구역', '블럭', '인벤토리', '시즌'];
    if (py > 60 && py < 100) {
      const tabW = this.canvas.width / tabs.length;
      const tabIdx = Math.floor(px / tabW);
      if (tabs[tabIdx]) {
        this.activeTab = tabs[tabIdx];
        this.scrollOffset = 0;
        return true;
      }
    }

    // Card buy button clicks
    const cards = PRODUCT_CARDS.filter(c => c.tab === this.activeTab);
    const cols = this.canvas.width >= 600 ? 3 : 2;
    const cardW = (this.canvas.width - 32 - (cols - 1) * 12) / cols;
    const cardH = 140;
    const startY = 116;

    cards.forEach((card, i) => {
      const col = i % cols;
      const row = Math.floor(i / cols);
      const x = 16 + col * (cardW + 12);
      const y = startY + row * (cardH + 12) - this.scrollOffset;
      const btnY = y + cardH - 36;
      if (px > x + 10 && px < x + cardW - 10 && py > btnY && py < btnY + 28) {
        this.handleBuyCard(card.productId);
      }
    });

    return true; // always consume when store is open
  }

  handleScroll(deltaY: number): void {
    if (!this.isOpen) return;
    this.scrollOffset = Math.max(0, this.scrollOffset + deltaY);
  }

  private showToast(msg: string, type: 'success' | 'error'): void {
    (window as any).__sandCastleToast?.(msg, type);
  }
}

export const storeScreen = new StoreScreen(
  document.querySelector<HTMLCanvasElement>('#game-canvas')!,
  document.querySelector<HTMLCanvasElement>('#game-canvas')!.getContext('2d')!
);
```

## Key Constraints
- The store overlay blocks all world input while open — `handlePointerDown` always returns `true`.
- Non-consumable products (`shovel_lv2/3/5`, `bucket_extra`, `plot_expand`, `inventory_expand`) permanently show "보유 중" once owned, regardless of session restarts (owned state is from `inventory.hydrate()`).
- `USER_CANCELLED` errors from the AIT SDK must display the cancellation message politely, not an error toast.
- The season tab in T50 will modify `PRODUCT_CARDS` filtering at store-open time — the `draw()` must re-read `PRODUCT_CARDS` each frame, not cache the filtered list.
- The loading spinner during purchase must prevent double-tapping the buy button (the `state.purchasing` guard covers this).
- Price labels are hard-coded display strings in Korean won format; the actual charge is processed by AIT and the values here are display-only.
- Scroll position resets to 0 on tab change to prevent invisible content when a shorter tab is selected.
