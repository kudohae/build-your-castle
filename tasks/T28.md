# T28 — [Phase 2] — Wind HUD & sand particles

## Overview
This task makes the wind system visually legible to players through two complementary elements: a directional arrow with a Korean strength label in the HUD, and a sand particle system that emits blowing particles during strong winds. Together they transform the invisible wind force into an evocative environmental effect — players can see which way the wind blows and how hard, and feel the desert storm atmosphere when강풍 strikes.

## Scope
Modify `src/ui/hud.ts` to render a wind indicator in the top HUD strip consisting of a canvas-drawn directional arrow (pointing in the current wind direction) and a Korean label text (`무풍`/`약풍`/`중풍`/`강풍`) positioned to its right. The arrow is drawn as a simple filled triangle (10px long, 5px wide) rotated to the wind direction using `ctx.rotate`. The label color is white for weak winds and orange-red for `강풍`. Modify `src/canvas/renderer.ts` to add a sand particle system: an array of particle objects `{ x, y, vx, vy, life, alpha }`. During `강풍` (strength ≥ 0.7), emit 5 new particles per frame from random x-positions within the BEACH zone (x: 0–800) at a y-position near the ground, with velocity components derived from wind direction/strength. Each particle lives for 45–75 frames, moves in the wind direction, and fades linearly from `alpha=1.0` to `0.0` over its lifetime. Particles are rendered as 2px circles in a tan/sand color (`#d4a857`). The array is capped at 300 particles; oldest particles are removed when exceeded.

## Output Files
- `src/ui/hud.ts (modify)` — adds `renderWindIndicator(ctx, wind, x, y)` called inside the HUD render function
- `src/canvas/renderer.ts (modify)` — adds particle array, `emitSandParticles(wind)`, `updateSandParticles(ctx)` called each frame

## Depends On

## Implementation Details

### Wind indicator arrow (HUD)

The arrow is drawn at a fixed canvas position in the top HUD strip, to the left of the tide countdown. It consists of a filled arrowhead pointing in the wind direction and a label string:

```typescript
// src/ui/hud.ts (addition)

import { getInterpolatedWind, Wind } from '../systems/weather';

const WIND_INDICATOR_X = 16; // left edge of HUD strip
const WIND_INDICATOR_Y = 56; // below stability gauge (added in T24)

function drawArrow(
  ctx: CanvasRenderingContext2D,
  cx: number,
  cy: number,
  angle: number, // radians
  color: string
): void {
  ctx.save();
  ctx.translate(cx, cy);
  ctx.rotate(angle);

  // Draw filled arrowhead pointing right (rotated to angle)
  ctx.fillStyle = color;
  ctx.beginPath();
  ctx.moveTo(10, 0);          // tip
  ctx.lineTo(-6, -5);         // left wing
  ctx.lineTo(-3, 0);          // base notch
  ctx.lineTo(-6, 5);          // right wing
  ctx.closePath();
  ctx.fill();

  // Arrow shaft
  ctx.strokeStyle = color;
  ctx.lineWidth = 2;
  ctx.beginPath();
  ctx.moveTo(-10, 0);
  ctx.lineTo(-3, 0);
  ctx.stroke();

  ctx.restore();
}

function windLabelColor(label: Wind['label']): string {
  switch (label) {
    case '무풍': return 'rgba(255,255,255,0.5)';
    case '약풍': return '#ffffff';
    case '중풍': return '#ffe082';
    case '강풍': return '#ff6d00';
    default: return '#ffffff';
  }
}

export function renderWindIndicator(
  ctx: CanvasRenderingContext2D,
  x: number,
  y: number
): void {
  const wind = getInterpolatedWind(new Date());
  const color = windLabelColor(wind.label);

  // Arrow pointing in wind direction
  drawArrow(ctx, x + 14, y, wind.direction, color);

  // Label text
  ctx.fillStyle = color;
  ctx.font = '12px "Apple SD Gothic Neo", sans-serif';
  ctx.textAlign = 'left';
  ctx.fillText(wind.label, x + 30, y + 5);

  // Strength percentage (for debugging / advanced display)
  if (wind.label === '강풍') {
    ctx.fillStyle = '#ff6d00';
    ctx.font = 'bold 11px "Apple SD Gothic Neo", sans-serif';
    ctx.fillText('🌬', x + 60, y + 5);
  }
}
```

### Sand particle data structure

```typescript
// src/canvas/renderer.ts (addition)

interface SandParticle {
  x: number;
  y: number;
  vx: number;
  vy: number;
  life: number;       // frames remaining
  maxLife: number;    // initial life for alpha calculation
  size: number;       // radius in px (1–2.5)
}

const sandParticles: SandParticle[] = [];
const MAX_SAND_PARTICLES = 300;
const SAND_COLOR = '#d4a857';
```

### Particle emission during 강풍

```typescript
import { getInterpolatedWind } from '../systems/weather';
import { BEACH_END_X, GROUND_Y } from '../types/zones';

function emitSandParticles(wind: ReturnType<typeof getInterpolatedWind>): void {
  if (wind.label !== '강풍') return; // only emit in strong wind

  const emitCount = 5;
  for (let i = 0; i < emitCount; i++) {
    if (sandParticles.length >= MAX_SAND_PARTICLES) {
      sandParticles.shift(); // evict oldest
    }

    const spawnX = Math.random() * BEACH_END_X; // random x in BEACH zone
    const spawnY = GROUND_Y - Math.random() * 60; // near ground surface

    const baseSpeed = wind.strength * 4; // max 4px/frame at full strength
    const jitter = (Math.random() - 0.5) * 1.5;

    sandParticles.push({
      x: spawnX,
      y: spawnY,
      vx: Math.cos(wind.direction) * baseSpeed + jitter,
      vy: Math.sin(wind.direction) * baseSpeed * 0.3 + jitter * 0.5,
      life: 45 + Math.floor(Math.random() * 31), // 45–75 frames
      maxLife: 75,
      size: 1 + Math.random() * 1.5,
    });
  }
}
```

### Particle update and render

```typescript
function updateSandParticles(
  ctx: CanvasRenderingContext2D,
  cameraX: number
): void {
  for (let i = sandParticles.length - 1; i >= 0; i--) {
    const p = sandParticles[i];

    // Update position
    p.x += p.vx;
    p.y += p.vy;
    p.life--;

    if (p.life <= 0) {
      sandParticles.splice(i, 1);
      continue;
    }

    const alpha = p.life / p.maxLife; // linear fade 1.0 → 0.0
    const screenX = p.x - cameraX;

    // Cull off-screen particles (no render, but still simulate)
    if (screenX < -10 || screenX > ctx.canvas.width + 10) continue;

    ctx.save();
    ctx.globalAlpha = alpha;
    ctx.fillStyle = SAND_COLOR;
    ctx.beginPath();
    ctx.arc(screenX, p.y, p.size, 0, Math.PI * 2);
    ctx.fill();
    ctx.restore();
  }
}
```

### Integration in renderFrame

```typescript
// src/canvas/renderer.ts — inside renderFrame:

const wind = getInterpolatedWind(new Date());

```

## Key Constraints
- `getInterpolatedWind` is called once per frame per render path — do not call it multiple times per frame (cache the result in a local variable).
- The sand particle array must never exceed `MAX_SAND_PARTICLES = 300`; use FIFO eviction (`shift()`) when full.
- Particles are world-space objects; subtract `cameraX` when rendering to convert to screen space.
- The wind arrow must use `ctx.rotate` + `ctx.translate` with `ctx.save()/restore()` to avoid affecting other HUD rendering.
- During `무풍` and `약풍`, the particle emitter emits nothing — do not render an empty loop.
- The label `강풍` should flash or stand out; use the orange color `#ff6d00` to signal danger.
