# T24 — [Phase 2] — Stability HUD

## Overview
This task adds a real-time structural stability gauge to the player's HUD, providing immediate visual feedback on how sound their sand castle is. The gauge transitions through a green-yellow-red color gradient, displays Korean-language danger warnings at low scores, triggers a CSS shake animation on the canvas when the structure becomes dangerous, and at zero stability invokes the physics collapse simulation. This is the primary player feedback loop for the entire structural gameplay system.

## Scope
Modify `src/ui/hud.ts` to add a stability gauge bar rendered on canvas below the existing top HUD strip. The bar is 200px wide and 12px tall, positioned at HUD top + 36px (one row below the resource/score line). The fill color transitions smoothly: green (`#4caf50`) for scores ≥ 60, yellow (`#ffb300`) for 30–59, and red (`#f44336`) for scores < 30. The transition is a linear interpolation between the two nearest colors — not a hard step — so the bar visually flows as scores change. A `"⚠ 위험"` label in 13px bold red is drawn to the right of the bar when the score is below 30. When score drops below 30, add a `shake` CSS class to the canvas element via `canvas.classList.add('hud-shake')` and remove it after 500ms (the CSS animation handles the motion). At score 0, call `triggerCollapse()` imported from `src/physics/engine.ts`, which makes all block bodies dynamic and applies random impulses. The HUD calls `calculateStability(blocks)` at most once per second (use a timestamp gate) and caches the last `StabilityResult` for rendering.

## Output Files
- `src/ui/hud.ts (modify)` — adds stability gauge bar, danger label, shake trigger, collapse trigger; adds `renderStabilityGauge(ctx, result)` internal function

## Implementation Details

### Color interpolation helper

```typescript
// src/ui/hud.ts (addition)

function lerpColor(a: string, b: string, t: number): string {
  // t in [0, 1]; 0 = color a, 1 = color b
  const parse = (hex: string) => {
    const h = hex.replace('#', '');
    return [parseInt(h.slice(0, 2), 16), parseInt(h.slice(2, 4), 16), parseInt(h.slice(4, 6), 16)];
  };
  const [ar, ag, ab] = parse(a);
  const [br, bg, bb] = parse(b);
  const r = Math.round(ar + (br - ar) * t);
  const g = Math.round(ag + (bg - ag) * t);
  const bl = Math.round(ab + (bb - ab) * t);
  return `#${r.toString(16).padStart(2, '0')}${g.toString(16).padStart(2, '0')}${bl.toString(16).padStart(2, '0')}`;
}

function stabilityColor(score: number): string {
  if (score >= 60) return '#4caf50'; // green
  if (score >= 30) {
    // yellow (score=60) → red (score=30): t = (60-score)/30
    const t = (60 - score) / 30;
    return lerpColor('#4caf50', '#ffb300', t);
  }
  // red zone: yellow (score=30) → deep red (score=0)
  const t = (30 - score) / 30;
  return lerpColor('#ffb300', '#f44336', t);
}
```

### Stability gauge render function

```typescript
const GAUGE_WIDTH = 200;
const GAUGE_HEIGHT = 12;
const GAUGE_X = 16; // left margin
const GAUGE_Y = 36; // below top strip

function renderStabilityGauge(
  ctx: CanvasRenderingContext2D,
  result: StabilityResult,
  canvas: HTMLCanvasElement
): void {
  const { score } = result;

  // Background track
  ctx.fillStyle = 'rgba(0,0,0,0.3)';
  ctx.beginPath();
  ctx.roundRect(GAUGE_X, GAUGE_Y, GAUGE_WIDTH, GAUGE_HEIGHT, 4);
  ctx.fill();

  // Fill bar
  const fillWidth = Math.max(0, (score / 100) * GAUGE_WIDTH);
  const color = stabilityColor(score);
  ctx.fillStyle = color;
  ctx.beginPath();
  ctx.roundRect(GAUGE_X, GAUGE_Y, fillWidth, GAUGE_HEIGHT, 4);
  ctx.fill();

  // Score label
  ctx.fillStyle = '#ffffff';
  ctx.font = '11px "Apple SD Gothic Neo", sans-serif';
  ctx.textAlign = 'left';
  ctx.fillText(`안정성 ${Math.round(score)}`, GAUGE_X, GAUGE_Y - 4);

  // Danger label
  if (score < 30) {
    ctx.fillStyle = '#f44336';
    ctx.font = 'bold 13px "Apple SD Gothic Neo", sans-serif';
    ctx.fillText('⚠ 위험', GAUGE_X + GAUGE_WIDTH + 8, GAUGE_Y + 10);
  }

  // Warning messages (show topmost critical warning)
  const criticalWarning = result.warnings.find((w) => w.severity === 'critical');
  if (criticalWarning) {
    ctx.fillStyle = '#ff5252';
    ctx.font = '11px "Apple SD Gothic Neo", sans-serif';
    ctx.fillText(criticalWarning.message, GAUGE_X, GAUGE_Y + GAUGE_HEIGHT + 14);
  }
}
```

```typescript
// Inside hud.ts — trigger shake when score drops below 30
let lastShakeScore = 100;

function maybeShakeCanvas(canvas: HTMLCanvasElement, score: number): void {
  if (score < 30 && lastShakeScore >= 30) {
    // Crossed below threshold
    canvas.classList.add('hud-shake');
    setTimeout(() => canvas.classList.remove('hud-shake'), 500);
  }
  lastShakeScore = score;
}
```

### Collapse trigger at zero

```typescript
import { triggerCollapse } from '../physics/engine';

let collapseFired = false;

function maybeCollapse(result: StabilityResult, blocks: Block[]): void {
  if (result.score === 0 && blocks.length > 0 && !collapseFired) {
    collapseFired = true;
    triggerCollapse();
    // Reset collapse flag after 3 seconds so a new castle can be built
    setTimeout(() => { collapseFired = false; }, 3000);
  }
}
```

### Throttled stability calculation

```typescript
let lastStabilityCalcTime = 0;
let lastStabilityResult: StabilityResult | null = null;
const STABILITY_CALC_INTERVAL_MS = 1000;

export function updateStabilityHUD(
  ctx: CanvasRenderingContext2D,
  canvas: HTMLCanvasElement,
  blocks: Block[]
): void {
  const now = Date.now();
  if (now - lastStabilityCalcTime >= STABILITY_CALC_INTERVAL_MS || !lastStabilityResult) {
    lastStabilityResult = calculateStability(blocks);
    lastStabilityCalcTime = now;
  }

  renderStabilityGauge(ctx, lastStabilityResult, canvas);
  maybeShakeCanvas(canvas, lastStabilityResult.score);
  maybeCollapse(lastStabilityResult, blocks);
}
```

Call `updateStabilityHUD(ctx, canvas, blocks)` from the main HUD render function each frame.

### triggerCollapse stub (for engine.ts)
This function must be exported from `src/physics/engine.ts`. If T08 has not yet implemented it, add the stub:

```typescript
// src/physics/engine.ts (addition)
import Matter from 'matter-js';

export function triggerCollapse(): void {
  const allBodies = Matter.Composite.allBodies(engine.world);
  for (const body of allBodies) {
    if (body.isStatic) continue;
    Matter.Body.applyForce(body, body.position, {
      x: (Math.random() - 0.5) * body.mass * 0.5,
      y: -Math.random() * body.mass * 0.3,
    });
  }
}
```

## Key Constraints
- `calculateStability` is expensive (O(blocks × 1024)); it must be throttled to at most 1 call/second.
- The shake animation must not be retriggered more than once per threshold crossing; use the `lastShakeScore` comparator pattern.
- `triggerCollapse` must be called at most once per collapse event; use the `collapseFired` guard.
- The gauge bar must use `roundRect` — if the browser does not support it, fall back to `fillRect`.
- The gauge is rendered on the canvas, not as a DOM element, to avoid z-index conflicts with the game canvas.
- The CSS `hud-shake` class is added to the `<canvas>` element, which must have `transform-origin: center` in its CSS.
