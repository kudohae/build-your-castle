# T23 — [Phase 2] — Stability calculator

## Overview
The stability calculator is the core scoring engine for a player's sand castle. It evaluates structural soundness across five dimensions: center-of-mass balance, ground support ratio, moisture bonus, carving penalty, and a center-of-mass bonus for well-balanced structures. The resulting score (0–100) drives the HUD gauge (T24), informs leaderboard ranking (T33), and at zero triggers the collapse simulation. Korean-language warning messages surface actionable feedback to the player in real time.

## Scope
Implement `calculateStability(blocks: Block[]): StabilityResult` in `src/physics/stability.ts`. The algorithm: (1) **centerOfMass**: compute `{ x: Σ(mass_i × x_i) / Σ(mass_i), y: Σ(mass_i × y_i) / Σ(mass_i) }` using each block's body mass and grid center coordinate. (2) **supportRatio**: count blocks with `block.y + BLOCK_SIZE >= GROUND_Y` (touching ground layer) divided by total block count. (3) **wetRatio**: count blocks with `block.isWet === true` divided by total. (4) **carvingPenalty**: compute average `pixelMask` value across all blocks (sum all 1024 values per block, divide by `1024 * blocks.length`); penalty = `1 - avgMaskValue` (a fully solid castle has penalty 0; a heavily carved castle approaches 1). (5) **centerOfMassBonus**: the horizontal CoM should be within the support footprint. Define `supportMinX` and `supportMaxX` from ground-touching blocks; if CoM.x is inside `[supportMinX, supportMaxX]`, bonus = `1 - (2 * |CoM.x - footprintCenter| / footprintWidth)`, else bonus = 0. (6) **score** = `clamp(supportRatio*50 + wetRatio*20 - carvingPenalty*30 + centerOfMassBonus*20, 0, 100)`. Populate `warnings[]` with Korean message strings for specific threshold violations.

### Center of mass calculation

```typescript
function computeCenterOfMass(blocks: Block[]): { x: number; y: number } {
  let totalMass = 0;
  let weightedX = 0;
  let weightedY = 0;

  for (const block of blocks) {
    const mass = block.body?.mass ?? 80; // fallback to dry_sand mass
    const cx = block.x + BLOCK_SIZE / 2;
    const cy = block.y + BLOCK_SIZE / 2;
    weightedX += mass * cx;
    weightedY += mass * cy;
    totalMass += mass;
  }

  if (totalMass === 0) return { x: 0, y: 0 };
  return { x: weightedX / totalMass, y: weightedY / totalMass };
}
```

### Support ratio and footprint

```typescript
function computeSupport(blocks: Block[]): {
  ratio: number;
  minX: number;
  maxX: number;
} {
  let groundCount = 0;
  let minX = Infinity;
  let maxX = -Infinity;

  for (const block of blocks) {
    if (block.y + BLOCK_SIZE >= GROUND_Y) {
      groundCount++;
      minX = Math.min(minX, block.x);
      maxX = Math.max(maxX, block.x + BLOCK_SIZE);
    }
  }

  return {
    ratio: blocks.length > 0 ? groundCount / blocks.length : 0,
    minX: isFinite(minX) ? minX : 0,
    maxX: isFinite(maxX) ? maxX : 0,
  };
}
```

### Carving penalty

```typescript
function computeCarvingPenalty(blocks: Block[]): number {
  if (blocks.length === 0) return 0;

  let totalMaskSum = 0;
  const pixelsPerBlock = BLOCK_SIZE * BLOCK_SIZE; // 1024

  for (const block of blocks) {
    let blockSum = 0;
    for (let i = 0; i < pixelsPerBlock; i++) {
      blockSum += block.pixelMask[i];
    }
    totalMaskSum += blockSum / pixelsPerBlock; // normalize to [0,1] per block
  }

  const avgMaskValue = totalMaskSum / blocks.length;
  return 1 - avgMaskValue; // 0 = fully solid (no penalty), 1 = fully carved
}
```

### Center-of-mass bonus

```typescript
function computeCoMBonus(
  com: { x: number },
  supportMinX: number,
  supportMaxX: number
): number {
  const footprintWidth = supportMaxX - supportMinX;
  if (footprintWidth <= 0) return 0;

  const footprintCenter = (supportMinX + supportMaxX) / 2;
  const offset = Math.abs(com.x - footprintCenter);
  const normalizedOffset = (2 * offset) / footprintWidth;

  // Bonus is 1 when perfectly centered, 0 when CoM is at the edge, negative beyond
  return Math.max(0, 1 - normalizedOffset);
}
```

### Main function and warnings

```typescript
function clamp(value: number, min: number, max: number): number {
  return Math.max(min, Math.min(max, value));
}

export function calculateStability(blocks: Block[]): StabilityResult {
  const warnings: StabilityWarning[] = [];

  if (blocks.length === 0) {
    return {
      score: 0,
      centerOfMass: { x: 0, y: 0 },
      supportRatio: 0,
      wetRatio: 0,
      carvingPenalty: 0,
      centerOfMassBonus: 0,
      warnings: [{ code: 'EMPTY', message: '블록을 배치해 주세요.', severity: 'info' }],
    };
  }

  const com = computeCenterOfMass(blocks);
  const { ratio: supportRatio, minX, maxX } = computeSupport(blocks);
  const wetRatio = blocks.filter((b) => b.isWet).length / blocks.length;
  const carvingPenalty = computeCarvingPenalty(blocks);
  const centerOfMassBonus = computeCoMBonus(com, minX, maxX);

  const score = clamp(
    supportRatio * 50 +
    wetRatio * 20 -
    carvingPenalty * 30 +
    centerOfMassBonus * 20,
    0,
    100
  );

  // === Warning generation ===
  if (supportRatio < 0.3) {
    warnings.push({
      code: 'LOW_SUPPORT',
      message: '지지대가 부족합니다. 바닥 블록을 더 놓아 주세요.',
      severity: 'warn',
    });
  }

  if (centerOfMassBonus < 0.3) {
    warnings.push({
      code: 'COM_TILT',
      message: '무게중심이 기울었습니다. 구조물을 좌우 균형 있게 쌓아 주세요.',
      severity: 'warn',
    });
  }

  if (com.x < minX || com.x > maxX) {
    warnings.push({
      code: 'COM_OUTSIDE',
      message: '⚠ 무게중심이 지지대 밖에 있습니다! 곧 무너질 수 있어요.',
      severity: 'critical',
    });
  }

  if (carvingPenalty > 0.6) {
    warnings.push({
      code: 'OVER_CARVED',
      message: '조각이 너무 많아 구조가 약해지고 있습니다.',
      severity: 'warn',
    });
  }

  if (wetRatio === 0 && blocks.length > 5) {
    warnings.push({
      code: 'DRY_CASTLE',
      message: '물을 부으면 모래가 더 단단해집니다!',
      severity: 'info',
    });
  }

  if (score < 10 && blocks.length > 0) {
    warnings.push({
      code: 'COLLAPSE_IMMINENT',
      message: '🚨 구조물이 무너지려 합니다!',
      severity: 'critical',
    });
  }

  return {
    score,
    centerOfMass: com,
    supportRatio,
    wetRatio,
    carvingPenalty,
    centerOfMassBonus,
    warnings,
  };
}
```

### Score formula reference table
| Condition | Points |
|---|---|
| All blocks touching ground | +50 |
| All blocks wet | +20 |
| CoM perfectly centered | +20 |
| All blocks fully carved | -30 |
| **Max possible** | **90** (wet + centered + full support, no carving) |
| **Min possible** | **0** (clamped) |

## Key Constraints
- `GROUND_Y` is the y-coordinate of the ground plane; blocks with `y + 32 >= GROUND_Y` are considered ground-touching.
- If a block has no `body` (e.g., it was just placed and physics hasn't initialized), fall back to the block type's default mass.
- `carvingPenalty` iterates all 1024 mask values per block — with 300 blocks this is 307,200 iterations per frame. Call `calculateStability` at most once per second (throttled by the HUD, not the calculator itself).
- Warnings are regenerated fresh on each call; do not cache or accumulate across calls.
- The function must not mutate any block data — it is a pure read.

## Acceptance Criteria
- [ ] A single dry_sand block touching the ground with a fully solid mask returns `score >= 50`.
- [ ] An empty block array returns `score === 0` and a single `'EMPTY'` warning.
- [ ] A fully wet castle (all blocks isWet) scores at least 20 points higher than the same structure dry.
- [ ] A castle with CoM outside the support footprint includes a `'COM_OUTSIDE'` warning with severity `'critical'`.
- [ ] A block array where all mask values are 0.5 (engrave) produces `carvingPenalty === 0.5` and score is reduced by 15 points vs the same all-solid structure.
- [ ] `calculateStability` does not mutate any block, body, or pixelMask.
- [ ] Score is always in `[0, 100]` regardless of extreme inputs.
- [ ] Korean warning messages are non-empty strings for all defined warning codes.
