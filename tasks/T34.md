# T34 — [Phase 3] — Leaderboard UI screen

## Overview
The leaderboard screen ("명예의 전당" — Hall of Fame) is the final social layer of the game, giving players a scrollable full-screen view of the top 20 sand castle builders ranked by community art scores. Medal icons distinguish the top three players, the current user's row is always visible and highlighted even outside the top 20, and a single "닫기" button returns the player to the game. This screen closes the game loop: build → get rated → see your rank → build better.

## Scope
Create `src/ui/leaderboard.ts` with `openLeaderboard()`, `closeLeaderboard()`, and `renderLeaderboard(ctx, canvas)` functions. `openLeaderboard` calls `getTopPlayers(20)` from T33 and `getCurrentUserRank(currentUserId)` to populate the list, then sets a module-level `isOpen` flag. `renderLeaderboard` draws a full-screen dark overlay (`rgba(0,0,0,0.85)`), a centered panel (max 420px wide), a title "🏰 명예의 전당 🏰", and a scrollable list of up to 20 rows. Each row: rank number (🥇/🥈/🥉 for rank 1/2/3, plain number otherwise), nickname (truncated to 16 chars), avg rating (1 decimal, e.g. "★ 4.2"), and vote count in parentheses ("(12표)"). The current user's row has a gold background highlight `rgba(255, 215, 0, 0.25)` and is additionally pinned at the bottom of the panel if not in the top 20. A "닫기" button (48×28px) in the top-right of the panel calls `closeLeaderboard`. Scrolling is handled by tracking `scrollOffsetY` modified by `wheel` and touch drag events on the canvas when the leaderboard is open. Add a "명예의 전당" button to `src/ui/hud.ts` at the top-right of the main HUD (top right, after the tide countdown) that calls `openLeaderboard()`.

## Output Files
- `src/ui/leaderboard.ts` — new file; exports `openLeaderboard`, `closeLeaderboard`, `renderLeaderboard`, `handleLeaderboardInput`
- `src/ui/hud.ts (modify)` — adds "명예의 전당" button in HUD top strip; registers click/tap handler that calls `openLeaderboard()`

### Module state

```typescript
// src/ui/leaderboard.ts

import { getTopPlayers, getCurrentUserRank, LeaderboardEntry } from '../ait/leaderboard';

interface LeaderboardState {
  isOpen: boolean;
  entries: LeaderboardEntry[];
  currentUserEntry: LeaderboardEntry | null;
  currentUserRank: number | null;
  scrollOffsetY: number;
  isLoading: boolean;
  currentUserId: string | null;
}

const state: LeaderboardState = {
  isOpen: false,
  entries: [],
  currentUserEntry: null,
  currentUserRank: null,
  scrollOffsetY: 0,
  isLoading: false,
  currentUserId: null,
};

const ROW_HEIGHT = 52;
const PANEL_WIDTH = 420;
const PANEL_PADDING = 20;
const VISIBLE_ROWS = 8; // rows visible without scrolling
const PANEL_HEIGHT = 60 + VISIBLE_ROWS * ROW_HEIGHT + 80; // header + rows + user pin
```

### openLeaderboard

```typescript
export async function openLeaderboard(currentUserId: string): Promise<void> {
  if (state.isLoading) return;

  state.isOpen = true;
  state.isLoading = true;
  state.scrollOffsetY = 0;
  state.currentUserId = currentUserId;
  state.entries = [];
  state.currentUserEntry = null;
  state.currentUserRank = null;

  try {
    const [entries, rank] = await Promise.all([
      getTopPlayers(20),
      getCurrentUserRank(currentUserId),
    ]);

    state.entries = entries;
    state.currentUserRank = rank;

    // Check if current user is in top 20
    const inTop20 = entries.some((e) => e.userId === currentUserId);
    if (!inTop20 && rank !== null) {
      // Build a placeholder entry for the pinned current-user row
      state.currentUserEntry = {
        rank: rank,
        userId: currentUserId,
        nickname: '나', // "Me" — could be replaced with actual nickname from profile
        score: 0,       // will be fetched separately if needed
        avgRating: 0,
        voteCount: 0,
      };
    } else {
      state.currentUserEntry = entries.find((e) => e.userId === currentUserId) ?? null;
    }
  } catch (err) {
    console.error('[Leaderboard] Failed to load:', err);
    state.entries = [];
  } finally {
    state.isLoading = false;
  }
}

export function closeLeaderboard(): void {
  state.isOpen = false;
  state.scrollOffsetY = 0;
}
```

### Rank medal helper

```typescript
function rankLabel(rank: number): string {
  switch (rank) {
    case 1: return '🥇';
    case 2: return '🥈';
    case 3: return '🥉';
    default: return String(rank).padStart(2, ' ');
  }
}

function rankLabelFontSize(rank: number): string {
  return rank <= 3 ? '20px' : '14px "SF Mono", monospace';
}
```

### renderLeaderboard

```typescript
export function renderLeaderboard(
  ctx: CanvasRenderingContext2D,
  canvasWidth: number,
  canvasHeight: number
): void {
  if (!state.isOpen) return;

  // Full-screen overlay
  ctx.fillStyle = 'rgba(0, 0, 0, 0.85)';
  ctx.fillRect(0, 0, canvasWidth, canvasHeight);

  const panelX = (canvasWidth - PANEL_WIDTH) / 2;
  const panelY = (canvasHeight - PANEL_HEIGHT) / 2;

  // Panel background
  ctx.fillStyle = '#1a1a2e';
  ctx.beginPath();
  ctx.roundRect(panelX, panelY, PANEL_WIDTH, PANEL_HEIGHT, 16);
  ctx.fill();
  ctx.strokeStyle = '#3d3d6b';
  ctx.lineWidth = 1.5;
  ctx.stroke();

  // Title
  ctx.fillStyle = '#f0c040';
  ctx.font = 'bold 20px "Apple SD Gothic Neo", sans-serif';
  ctx.textAlign = 'center';
  ctx.fillText('🏰 명예의 전당 🏰', canvasWidth / 2, panelY + 38);

  // Close button
  renderCloseButton(ctx, panelX + PANEL_WIDTH - 16, panelY + 16);

  if (state.isLoading) {
    ctx.fillStyle = '#aaaaaa';
    ctx.font = '14px "Apple SD Gothic Neo", sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText('불러오는 중...', canvasWidth / 2, panelY + 100);
    return;
  }

  if (state.entries.length === 0) {
    ctx.fillStyle = '#888888';
    ctx.textAlign = 'center';
    ctx.fillText('아직 등록된 플레이어가 없습니다', canvasWidth / 2, panelY + 100);
    return;
  }

  // Scrollable rows clip region
  const listStartY = panelY + 56;
  const listHeight = VISIBLE_ROWS * ROW_HEIGHT;

  ctx.save();
  ctx.beginPath();
  ctx.rect(panelX, listStartY, PANEL_WIDTH, listHeight);
  ctx.clip();

  state.entries.forEach((entry, i) => {
    const rowY = listStartY + i * ROW_HEIGHT - state.scrollOffsetY;
    renderRow(ctx, entry, panelX, rowY, PANEL_WIDTH, state.currentUserId);
  });

  ctx.restore();

  // Pinned current user row at bottom (if not in top 20)
  const isInTop20 = state.entries.some((e) => e.userId === state.currentUserId);
  if (!isInTop20 && state.currentUserEntry) {
    const pinnedY = panelY + PANEL_HEIGHT - ROW_HEIGHT - PANEL_PADDING;
    // Divider
    ctx.strokeStyle = '#444466';
    ctx.lineWidth = 1;
    ctx.beginPath();
    ctx.moveTo(panelX + PANEL_PADDING, pinnedY - 4);
    ctx.lineTo(panelX + PANEL_WIDTH - PANEL_PADDING, pinnedY - 4);
    ctx.stroke();

    renderRow(ctx, state.currentUserEntry, panelX, pinnedY, PANEL_WIDTH, state.currentUserId);
  }
}
```

### renderRow helper

```typescript
function renderRow(
  ctx: CanvasRenderingContext2D,
  entry: LeaderboardEntry,
  panelX: number,
  rowY: number,
  panelWidth: number,
  currentUserId: string | null
): void {
  const isCurrentUser = entry.userId === currentUserId;
  const rowH = ROW_HEIGHT - 4;

  // Row background
  if (isCurrentUser) {
    ctx.fillStyle = 'rgba(255, 215, 0, 0.25)';
    ctx.beginPath();
    ctx.roundRect(panelX + 8, rowY + 2, panelWidth - 16, rowH, 6);
    ctx.fill();
  } else if (entry.rank <= 3) {
    ctx.fillStyle = 'rgba(255, 215, 0, 0.08)';
    ctx.beginPath();
    ctx.roundRect(panelX + 8, rowY + 2, panelWidth - 16, rowH, 6);
    ctx.fill();
  }

  const textY = rowY + ROW_HEIGHT / 2 + 5;

  // Rank
  ctx.fillStyle = entry.rank <= 3 ? '#f0c040' : '#888888';
  ctx.font = entry.rank <= 3 ? '20px serif' : '14px "SF Mono", monospace';
  ctx.textAlign = 'left';
  ctx.fillText(rankLabel(entry.rank), panelX + 16, textY);

  // Nickname (truncated)
  const nick = entry.nickname.length > 16
    ? entry.nickname.slice(0, 15) + '…'
    : entry.nickname;
  ctx.fillStyle = isCurrentUser ? '#ffd700' : '#e0e0e0';
  ctx.font = isCurrentUser
    ? 'bold 14px "Apple SD Gothic Neo", sans-serif'
    : '14px "Apple SD Gothic Neo", sans-serif';
  ctx.fillText(nick, panelX + 56, textY);

  // Avg rating
  ctx.fillStyle = '#ffc107';
  ctx.font = '13px "Apple SD Gothic Neo", sans-serif';
  ctx.textAlign = 'right';
  ctx.fillText(`★ ${entry.avgRating.toFixed(1)}`, panelX + panelWidth - 80, textY);

  // Vote count
  ctx.fillStyle = '#888888';
  ctx.font = '12px "Apple SD Gothic Neo", sans-serif';
  ctx.fillText(`(${entry.voteCount}표)`, panelX + panelWidth - 16, textY);
}
```

### Scroll and input handling

```typescript
export function handleLeaderboardInput(
  e: WheelEvent | PointerEvent,
  canvasWidth: number,
  canvasHeight: number
): boolean {
  if (!state.isOpen) return false;

  if (e instanceof WheelEvent) {
    const maxScroll = Math.max(0, state.entries.length * ROW_HEIGHT - VISIBLE_ROWS * ROW_HEIGHT);
    state.scrollOffsetY = Math.max(0, Math.min(maxScroll, state.scrollOffsetY + e.deltaY));
    return true; // consumed
  }

  if (e instanceof PointerEvent && e.type === 'pointerdown') {
    const panelX = (canvasWidth - PANEL_WIDTH) / 2;
    const panelY = (canvasHeight - PANEL_HEIGHT) / 2;
    const closeBtnX = panelX + PANEL_WIDTH - 16 - 48;
    const closeBtnY = panelY + 16;

    if (
      e.clientX >= closeBtnX && e.clientX <= closeBtnX + 48 &&
      e.clientY >= closeBtnY && e.clientY <= closeBtnY + 28
    ) {
      closeLeaderboard();
    }
    return true; // consumed — prevent click-through to game
  }

  return false;
}
```

## Key Constraints
- `renderLeaderboard` must be called every frame while `isOpen === true` and must completely block interaction with the game canvas beneath it (all pointer events return `true` from `handleLeaderboardInput` to signal consumption).
- The scroll clip region must use `ctx.save()/ctx.clip()/ctx.restore()` to prevent rows from overflowing the panel.
- Emoji rank medals (🥇🥈🥉) require the `serif` or `system-ui` font to render correctly on mobile; do not use a custom webfont for these.
- `openLeaderboard` must be idempotent — calling it while already open refreshes the data.
- The pinned current-user row is only shown when the user is **outside** the top 20; if they are in the list, their row is highlighted inline.
- Nickname truncation at 16 characters with an ellipsis prevents overflow into the score column.

## Acceptance Criteria
- [ ] Tapping "명예의 전당" in the HUD opens the leaderboard overlay and covers the game canvas.
- [ ] Top 20 entries are displayed in rank order with correct rank medals for positions 1–3.
- [ ] The current user's row is highlighted gold regardless of whether they are in the top 20.
- [ ] If the current user is outside the top 20, their row is pinned at the bottom of the panel.
- [ ] Scrolling the list (mouse wheel or touch drag) correctly scrolls the row list without moving the panel background or title.
- [ ] Tapping "닫기" closes the overlay and returns to the game.
- [ ] Nicknames longer than 16 characters are truncated with "…" in the row display.
- [ ] Pointer events are fully consumed by the leaderboard while open — game blocks underneath do not receive tap events.
