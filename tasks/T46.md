# T46 — [Phase 4] — Quest UI Screen

## Overview
This task implements the quest screen overlay and the celebration sequence that plays when a quest is completed. It provides players with a clear view of their progress toward all seven quests and rewards completion with a memorable full-screen confetti burst and a titled toast notification.

## Scope
Create `src/ui/quests.ts`. A full-screen overlay opens via the "도전 과제" HUD button and displays a scrollable 2-column grid of 7 quest cards. Each card shows: an emoji icon matched to the quest theme, the Korean title, a description of the win condition, a progress indicator (numeric or bar where applicable), the reward amount, and a ✅ (complete) or 🔒 (locked) status badge. Completion status is loaded from `getCompletedQuestIds()` (T45) when the screen opens. Separately, the module listens for the `quest:completed` CustomEvent (dispatched by T45) and triggers a full-screen celebration: a golden confetti particle system (50 particles with physics) animates for 3 seconds, and a centered toast reads "🎉 [칭호명] 칭호 획득!" for 3 seconds. Earned quest titles are stored in a `Set<string>` local state variable and rendered as a small colored badge next to the player's nickname in the HUD.

## Output Files
- `src/ui/quests.ts` — new file; `QuestScreen` class, confetti particle system, HUD badge renderer, `quest:completed` listener

## Depends On
- T45 — provides `getCompletedQuestIds()`, `QUESTS` (re-exported), and the `quest:completed` CustomEvent contract

```typescript
import { QUESTS, gameState, Quest } from '../systems/quests';
import { getCompletedQuestIds } from '../ait/promotion';

const QUEST_ICONS: Record<string, string> = {
  first_dig: '⛏️',
  tower:     '🗼',
  arch:      '🌉',
  sculptor:  '🗿',
  survivor:  '🌊',
  collab:    '👥',
  high_art:  '⭐',
};

const QUEST_TITLE_BADGES: Record<string, { label: string; color: string }> = {
  first_dig: { label: '개척자',    color: '#8bc34a' },
  tower:     { label: '하늘지기',  color: '#2196f3' },
  arch:      { label: '아치 장인', color: '#9c27b0' },
  sculptor:  { label: '조각가',   color: '#ff9800' },
  survivor:  { label: '파도 전사', color: '#03a9f4' },
  collab:    { label: '협업 왕',   color: '#e91e63' },
  high_art:  { label: '예술 감독', color: '#ffd700' },
};

function getQuestProgress(quest: Quest): { current: number; max: number } {
  const s = gameState;
  switch (quest.id) {
    case 'first_dig': return { current: Math.min(s.totalBlocksCollected, 1), max: 1 };
    case 'tower':     return { current: Math.min(s.maxHeight, 20), max: 20 };
    case 'arch':      return { current: 0, max: 1 }; // binary — no partial progress
    case 'sculptor':  return { current: Math.min(s.totalEngravedPixels, 100), max: 100 };
    case 'survivor':  return { current: Math.min(s.tidesWeathered, 3), max: 3 };
    case 'collab':    return { current: Math.min(s.collaboratorCount, 3), max: 3 };
    case 'high_art':  return { current: Math.min(s.voteCount, 3), max: 3 };
    default:          return { current: 0, max: 1 };
  }
}
```

### QuestScreen Class

```typescript
export class QuestScreen {
  private isOpen = false;
  private scrollOffset = 0;
  private completedIds: Set<string> = new Set();
  private earnedTitles: Set<string> = new Set();

  constructor(
    private canvas: HTMLCanvasElement,
    private ctx: CanvasRenderingContext2D
  ) {
    window.addEventListener('quest:completed', this.handleQuestCompleted.bind(this) as EventListener);
  }

  async open(): Promise<void> {
    this.isOpen = true;
    this.scrollOffset = 0;
    try {
      const { getCurrentUserId } = await import('../ait/auth');
      this.completedIds = await getCompletedQuestIds(getCurrentUserId());
    } catch {
      this.completedIds = new Set();
    }
  }

  close(): void {
    this.isOpen = false;
  }

  /** Called from the main render loop every frame. */
  draw(): void {
    if (!this.isOpen) return;
    const { ctx, canvas } = this;
    const W = canvas.width;
    const H = canvas.height;

    // Background
    ctx.fillStyle = 'rgba(20, 12, 4, 0.94)';
    ctx.fillRect(0, 0, W, H);

    // Header
    ctx.fillStyle = '#ffd700';
    ctx.font = 'bold 22px Pretendard, sans-serif';
    ctx.textAlign = 'center';
    ctx.fillText('🏆 도전 과제', W / 2, 44);
    ctx.textAlign = 'left';

    // Close button
    ctx.fillStyle = 'rgba(255,255,255,0.12)';
    ctx.beginPath();
    ctx.roundRect(W - 52, 14, 40, 36, 8);
    ctx.fill();
    ctx.fillStyle = '#fff';
    ctx.font = '20px sans-serif';
    ctx.fillText('✕', W - 38, 38);

    // Quest grid
    ctx.save();
    ctx.beginPath();
    ctx.rect(0, 64, W, H - 64);
    ctx.clip();
    this.drawQuestGrid();
    ctx.restore();

    // Confetti (drawn on top of everything)
    confettiSystem.draw(ctx);
  }

  private drawQuestGrid(): void {
    const { ctx, canvas } = this;
    const cols = 2;
    const cardW = (canvas.width - 48) / cols;
    const cardH = 150;
    const startY = 72;

    QUESTS.forEach((quest, i) => {
      const col = i % cols;
      const row = Math.floor(i / cols);
      const x = 16 + col * (cardW + 16);
      const y = startY + row * (cardH + 12) - this.scrollOffset;
      this.drawQuestCard(quest, x, y, cardW, cardH);
    });
  }

  private drawQuestCard(
    quest: Quest,
    x: number,
    y: number,
    w: number,
    h: number
  ): void {
    const { ctx } = this;
    const isDone = this.completedIds.has(quest.id);
    const progress = getQuestProgress(quest);
    const icon = QUEST_ICONS[quest.id] ?? '🎯';

    // Card background
    ctx.fillStyle = isDone ? 'rgba(80, 180, 80, 0.18)' : 'rgba(255,255,255,0.07)';
    ctx.beginPath();
    ctx.roundRect(x, y, w, h, 10);
    ctx.fill();

    // Status badge (top right of card)
    ctx.font = '22px sans-serif';
    ctx.textAlign = 'right';
    ctx.fillText(isDone ? '✅' : '🔒', x + w - 10, y + 28);
    ctx.textAlign = 'left';

    // Icon + Title
    ctx.font = '26px sans-serif';
    ctx.fillText(icon, x + 10, y + 34);

    ctx.fillStyle = isDone ? '#8de08d' : '#fff';
    ctx.font = `bold 14px Pretendard, sans-serif`;
    ctx.fillText(quest.title, x + 46, y + 28);

    // Description
    ctx.fillStyle = '#bbb';
    ctx.font = '11px Pretendard, sans-serif';
    const descWords = quest.description;
    ctx.fillText(descWords, x + 10, y + 52, w - 20);

    // Progress bar
    if (progress.max > 1 || quest.id === 'sculptor') {
      const barX = x + 10;
      const barY = y + 66;
      const barW = w - 20;
      const fill = Math.min(progress.current / progress.max, 1);

      ctx.fillStyle = 'rgba(255,255,255,0.12)';
      ctx.beginPath();
      ctx.roundRect(barX, barY, barW, 8, 4);
      ctx.fill();

      ctx.fillStyle = isDone ? '#4caf50' : '#f5a623';
      ctx.beginPath();
      ctx.roundRect(barX, barY, barW * fill, 8, 4);
      ctx.fill();

      ctx.fillStyle = '#ccc';
      ctx.font = '10px Pretendard, sans-serif';
      ctx.fillText(`${progress.current} / ${progress.max}`, barX, barY + 20);
    }

    // Reward preview
    ctx.fillStyle = '#ffd700';
    ctx.font = '11px Pretendard, sans-serif';
    ctx.fillText(`🎁 +${quest.reward}P`, x + 10, y + h - 12);

    // Title badge preview
    const badge = QUEST_TITLE_BADGES[quest.id];
    if (badge) {
      ctx.fillStyle = isDone ? badge.color : 'rgba(150,150,150,0.5)';
      ctx.beginPath();
      ctx.roundRect(x + 60, y + h - 26, 70, 18, 9);
      ctx.fill();
      ctx.fillStyle = isDone ? '#fff' : '#666';
      ctx.font = 'bold 10px Pretendard, sans-serif';
      ctx.textAlign = 'center';
      ctx.fillText(badge.label, x + 95, y + h - 12);
      ctx.textAlign = 'left';
    }
  }

  handlePointerDown(px: number, py: number): boolean {
    if (!this.isOpen) return false;
    const W = this.canvas.width;
    if (px > W - 52 && py > 14 && py < 50) {
      this.close();
      return true;
    }
    return true; // consume all taps while open
  }

  handleScroll(deltaY: number): void {
    if (!this.isOpen) return;
    const cols = 2;
    const rows = Math.ceil(QUESTS.length / cols);
    const maxScroll = Math.max(0, rows * 162 - (this.canvas.height - 80));
    this.scrollOffset = Math.max(0, Math.min(this.scrollOffset + deltaY, maxScroll));
  }

  private handleQuestCompleted(e: CustomEvent): void {
    const { questId, title } = e.detail;
    this.completedIds.add(questId);
    this.earnedTitles.add(questId);
    confettiSystem.burst();
    showQuestToast(title);
  }
}
```

### Confetti Particle System

```typescript
interface Particle {
  x: number; y: number;
  vx: number; vy: number;
  color: string;
  size: number;
  rotation: number;
  rotationSpeed: number;
  life: number;     // 0–1, decreasing
  decay: number;
}

const CONFETTI_COLORS = [
  '#ffd700', '#ff6b6b', '#4fc3f7', '#81c784', '#ce93d8', '#ffb74d'
];

class ConfettiSystem {
  private particles: Particle[] = [];

  burst(): void {
    const W = window.innerWidth;
    const H = window.innerHeight;
    for (let i = 0; i < 50; i++) {
      const angle = (Math.random() * Math.PI * 2);
      const speed = 4 + Math.random() * 8;
      this.particles.push({
        x: W / 2 + (Math.random() - 0.5) * 200,
        y: H / 3,
        vx: Math.cos(angle) * speed,
        vy: Math.sin(angle) * speed - 5, // upward bias
        color: CONFETTI_COLORS[Math.floor(Math.random() * CONFETTI_COLORS.length)],
        size: 6 + Math.random() * 8,
        rotation: Math.random() * Math.PI * 2,
        rotationSpeed: (Math.random() - 0.5) * 0.2,
        life: 1,
        decay: 0.012 + Math.random() * 0.008,
      });
    }
  }

  draw(ctx: CanvasRenderingContext2D): void {
    for (let i = this.particles.length - 1; i >= 0; i--) {
      const p = this.particles[i];
      p.x += p.vx;
      p.vy += 0.3; // gravity
      p.y += p.vy;
      p.rotation += p.rotationSpeed;
      p.life -= p.decay;

      if (p.life <= 0) {
        this.particles.splice(i, 1);
        continue;
      }

      ctx.save();
      ctx.globalAlpha = p.life;
      ctx.translate(p.x, p.y);
      ctx.rotate(p.rotation);
      ctx.fillStyle = p.color;
      ctx.fillRect(-p.size / 2, -p.size / 2, p.size, p.size * 0.6);
      ctx.restore();
    }
    ctx.globalAlpha = 1;
  }
}

const confettiSystem = new ConfettiSystem();
```

### Quest Completion Toast

```typescript
function showQuestToast(questTitle: string): void {
  const badge = Object.values(QUEST_TITLE_BADGES).find(
    (_, i) => QUESTS[i]?.title === questTitle
  )?.label ?? questTitle;

  (window as any).__sandCastleToast?.(
    `🎉 ${badge} 칭호 획득!`,
    'quest',
    3000
  );
}
```

### HUD Badge Renderer

Called from the HUD draw function to show earned title badges next to the nickname:

```typescript
export function drawEarnedBadges(
  ctx: CanvasRenderingContext2D,
  questScreen: QuestScreen,
  nickX: number,
  nickY: number
): void {
  const earned = Array.from((questScreen as any).earnedTitles as Set<string>);
  if (earned.length === 0) return;

  // Show the most recently earned badge
  const latestId = earned[earned.length - 1];
  const badge = QUEST_TITLE_BADGES[latestId];
  if (!badge) return;

  const badgeX = nickX + 100;
  ctx.fillStyle = badge.color;
  ctx.beginPath();
  ctx.roundRect(badgeX, nickY - 16, 70, 18, 9);
  ctx.fill();
  ctx.fillStyle = '#fff';
  ctx.font = 'bold 10px Pretendard, sans-serif';
  ctx.textAlign = 'center';
  ctx.fillText(badge.label, badgeX + 35, nickY - 2);
  ctx.textAlign = 'left';
}

export const questScreen = new QuestScreen(
  document.querySelector<HTMLCanvasElement>('#game-canvas')!,
  document.querySelector<HTMLCanvasElement>('#game-canvas')!.getContext('2d')!
);
```

## Key Constraints
- The confetti system runs on the game canvas, not a separate DOM element — `confettiSystem.draw()` must be called each frame in the render loop while particles exist, even when the quest screen is closed.
- `earnedTitles` persists across open/close cycles of the screen — it is initialized from `completedIds` fetched at `open()` time and grows as new completions arrive via CustomEvent.
- The `quest:completed` listener is attached in the constructor and is never removed (the screen is a singleton) — this is acceptable because there will be exactly one `QuestScreen` instance.
- Progress bars for the `first_dig` and `arch` quests are binary (0 or 1) — do not show a progress bar for these; show only ✅/🔒.
- The HUD badge shows only the most recently earned title; it does not cycle through all earned titles in this task.
- Confetti `burst()` fires once per completion event, producing exactly 50 new particles (they may overlap with residual particles from a prior burst).
- All canvas text renders use `Pretendard, sans-serif` for consistency with the rest of the UI.
